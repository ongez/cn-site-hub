# cn-site-hub

本项目是一个用于归档和发布多个独立 HTML 页面的仓库。页面内容不针对任何特定域名或网站，仅作为静态资源的集中存放与展示。

## 项目简介

cn-site-hub 旨在收集、整理并托管各类独立的 HTML 页面。这些页面可能来源于不同的场景或功能需求，统一存放于此，便于访问、维护与版本管理。

## 目录说明

仓库根目录下按功能或主题划分文件夹，每个文件夹内包含一个或多个 HTML 文件及其相关资源（如 CSS、JavaScript、图片等）。目录结构示例如下：

```
/
├── page-group-a/
│   ├── index.html
│   └── assets/
├── page-group-b/
│   └── ...
└── README.md
```

- 每个文件夹代表一类页面集合。
- 建议在文件夹内添加简短的说明文件（如 `README.md`），描述该组页面的用途。

## 页面归档说明

- 所有页面均为纯前端实现，不依赖后端服务。
- 页面内不应包含任何形式的追踪、广告或外部请求（除非必要且明确标注）。
- 归档时请保持页面独立性与完整性，尽量将资源内联或放置于同目录下。
- 新增页面请遵循现有目录结构，或在根目录下新建文件夹。

## 维护说明

- 欢迎提交 Pull Request 添加新的 HTML 页面，或改进现有页面。
- 提交前请确保页面无恶意代码、无外部依赖泄露风险。
- 仓库维护者会定期检查页面质量，对不符合归档规范的页面可能会进行调整或移除。
- 如有任何疑问或建议，请通过 Issues 进行讨论。
